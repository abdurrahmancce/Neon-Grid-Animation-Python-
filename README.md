# 🌐 Neon 3D Grid Animation (Python + OpenGL)

A visually engaging **3D neon grid animation** built using Python.
This project uses real-time rendering with OpenGL to simulate wave motion and depth, creating a smooth and dynamic visual experience.

---

## 🚀 Features

* 🌊 Smooth wave-based animation using sine and cosine functions
* 🎮 Real-time rendering with OpenGL
* 🔄 Continuous grid rotation for a 3D effect
* ✨ Neon-style grid lines
* ⚡ Lightweight and fast performance

---

## 🛠️ Technologies Used

* Python
* Pygame
* PyOpenGL

## 🧠 How It Works

* The grid is generated using a **2D coordinate system**
* Wave motion is created using:

  * `sin()` for horizontal variation
  * `cos()` for vertical variation
* These values simulate a **Z-axis (depth)** in 3D space
* OpenGL renders the grid using **GL_LINES**
* Continuous rotation adds a **dynamic perspective**
